		- Ansible Architecture & Components :::
		
		- Ansible controller 
			- Ansible_Nodes

			- Ansible Controller 
			- Ansible Nodes 
			- Modules 
			- Inventory File
				- Contains the details of the target servers 
					- Host Name 
					- User_Name 
					- Credentials details 
				- Here, the target servers can be maintained as groups 
					Dev_servers
						- Dev_Server1,2,3,4,5,........,100
					Test_servers
						- Test_Server1,2,3,4,5
					
					
			- Ansible.config file 
			
			- using SSH Ansible Controller connect to Ansible Nodes.


		
		
		
		- Core concepts :
		
		- Ansible Adhoc Commands :::
			- Used to execute single module at a time
			- This cannot be reused.
			
				Eg.: Install Git :
				
					- apt install git 
				
		
		- Ansible Playbooks :::
		
			- Used to execute series of modules on the target server 
			- Playbooks are written using *.yaml scripts 
			- Playbooks should be reuse
		
			- 	Eg.: Install Nginx :
			
					- apt install nginx 
					- Start nginx 
					
			- 	Eg.: Install MySQL Database :
			
					- Install Pre-requisites
					- Install MySQL_Server 
					- Start MySQL_Server
					- Create Database
